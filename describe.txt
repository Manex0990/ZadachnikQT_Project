ОПИСАНИЕ ПРОЕКТА.
Тема проекта: Математический задачник. Проект будет выполнен в виде приложения, ориентированного для компьютера.
Мой проект предназначен для подготовки к экзаменам и отработки вычислительных навыков из школьной программы.
В этом приложении будут генерироваться различные математические примеры, уравнения и т д.
А также будет тестеровщик приложения, который будет проверять правильность выполнения пользователем заданий.
При запуске приложения сразу появится главное меню, где будут кнопки, при нажатии на них будут генерироваться соответсвующие надписям на кнопках задания.
За решение заданий будут давать очки рейтинга, рейтинг также будет показан в главном меню. Задания будут нескольких уровней сложности: от самых легких до достаточно сложных.
Чем сложнее задание, тем больше за его решение можно получить рейтинга. 
По мимо обычных заданий в приложении будут тесты, состоящие из нескольких заданий, они тоже будут разные по сложности.
За тесты можно получить больше очков рейтинга, чем за обычные задания. Рейтинг за полностью верный тест фиксированный, а если есть неверно решенные задания, то начисляет частичный балл.
